# PixelWorldProxy
Simple Pixel World Proxy made in Rust.

# Usage
Clone the repository:
```
git clone https://github.com/zKevz/PixelWorldProxy
cd PixelWorldProxy
```

Usage is as simple as just
```rust
cargo run
```
